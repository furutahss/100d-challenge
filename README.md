# 100 Days of Code - Vibe Coding Challenge

SNSでの発信と技術力の向上、および日常の省力化を目的とした、100日間連続でプロジェクトを作成するチャレンジです。
実装はすべてAIを活用した「バイブコーディング」で行い、1日1プロジェクトのペースで完成させます。

## リポジトリの運用方針

- **モノレポ構成**: すべてのプロジェクトをこの単一リポジトリ内の個別ディレクトリ（例: `day001/`）で管理します。
- **個別ドキュメント**: 各プロジェクトのディレクトリ内に専用の `README.md` を配置し、仕様、実行方法、および実際に公開・デプロイしたサイト等のリンクを記載します。
- **独立性の担保**: 後からの切り出し（スピンオフ）を容易にするため、プロジェクト間での共通UIコンポーネントの使い回しは原則行いません。
