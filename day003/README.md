# どっちが大きい？直感タップ

2つの数字・りんご・図形から大きいほうを素早くタップする、子ども向けの知育ゲームです。全10問を遊び、正解数に応じたメッセージを表示します。

## 特徴

- 数字の大小、物の数、図形の大きさをランダムに出題
- 大きなタップ領域と読みやすいひらがな表記で、スマホ・タブレットに対応
- 外部ライブラリやビルド工程が不要な静的サイト
- 画面のセーフエリアと横向き表示にも対応

## 実行方法

`day003/index.html` をブラウザで開くか、リポジトリルートで次を実行します。

```bash
python3 -m http.server 8000 --directory day003
```

`http://localhost:8000` にアクセスします。

## 公開・デプロイ

`day003` 内のファイルをそのままXServerの公開ディレクトリへアップロードできます。サーバー側の処理や依存関係は必要ありません。
